Структура: Продуктовая модель (Трайбы и Сквады). Команда разработки «Пульса» входит в блок «HR-тех».

Функционал разработчика: Проектирование UI-кита, реализация бизнес-логики на React Native, оптимизация производительности мобильного приложения, интеграция с внутренними API.

Цифровые технологии: * Стек: React Native, TypeScript, React: react-query.

Инфраструктура: Внутреннее облако SberCloud, Jenkins/Bitbucket для CI/CD.

Архитектура: Микросервисы, использование паттернов для обеспечения отказоустойчивости при нагрузке в 300к+ сотрудников.
